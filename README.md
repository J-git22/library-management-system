# Library Management System

An Enterprise Library Management System using PHP (OOP + MVC), MySQL, HTML, CSS, JavaScript, and Bootstrap 5.

## Environment
- PHP 8.2+
- XAMPP / Apache / MySQL

## Setup
1. Point your Apache DocumentRoot to the `public/` directory or access via XAMPP alias.
2. Run `composer dump-autoload` to generate the autoloader.
3. Configure database in `app/config/config.php`.
